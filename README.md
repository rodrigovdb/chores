# README

## Flatpickr

Importmap
```sh
$ ./bin/importmap pin flatpickr stimulus-flatpickr@beta --download
```

Then, add to the layout
```ruby
<link rel="stylesheet" href="https://ga.jspm.io/npm:flatpickr@4.6.9/dist/flatpickr.min.css">
```
